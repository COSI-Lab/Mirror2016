Arch Linux

http://mirror.clarkson.edu/archlinux/iso/latest/

http://mirror.clarkson.edu/archlinux/

Upstream Mirror:
mirrors.rit.edu
Homepage:
http://www.archlinux.org/

Setup

To setup Arch to use our Mirror, open up /etc/pacman.d/mirrorlist and add the following line:

Server = http://mirror.clarkson.edu/archlinux/$repo/os/$arch

Rsync:
rsync://mirror.clarkson.edu/debian

Upstream Mirrors:
mirrors.rit.edu & cdimage.debian.org
Homepage:
http://www.debian.org/

Setup

To setup Debian to use our Mirror, open up /etc/apt/sources.list and modify it to have the following lines:

deb  http://mirror.clarkson.edu/debian stretch main
deb-src  http://mirror.clarkson.edu/debian stretch main

deb  http://mirror.clarkson.edu/debian stretch-updates main
deb-src  http://mirror.clarkson.edu/debian stretch-updates main

deb http://mirror.clarkson.edu/debian-security stretch/updates main
deb-src http://mirror.clarkson.edu/debian-security stretch/updates main
